## What is EasyHandy?
듣지못하는 청각 장애인 분들과 소통 하는 방법중 가장 대표적인 수화 🤙.  
직접 동영상을 시청하며 수화를 배울수 있으며, 당신의 손 모양이 올바른지 자동으로 인식해줍니다!
